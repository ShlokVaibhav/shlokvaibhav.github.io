Some good resources

First on the list is Notion, as per wiki, Notion is an application that provides components such as databases, kanban boards, wikis, calendars and reminders. The edge that notion has over traditional knowledge management softwares like one-note and evernote is that it allows us to treat each page as a class to which we can assign attributes like date, tags, urls and much more, we can also then do filtering based on these class attributes.
